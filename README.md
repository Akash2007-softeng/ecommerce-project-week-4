#  E-commerce Web Application
 Project Overview
This project is a simple E-commerce Web Application developed as part of a 4-week internship program.  
The application allows users to browse products, add them to a cart, calculate total price, and place orders.  
It also includes a basic login and signup system.
 Features:
-  Product Listing (multiple items)
-  Add to Cart functionality
-  Dynamic Total Price Calculation
-  User Login and Signup system
-  Welcome message after login
-  Order Placement system
-  Clear Cart option
-  Simple and clean user interface
## Technologies Used
- HTML (Structure)
- CSS (Styling)
- JavaScript (Functionality)
- Browser Local Storage (User Data)
## How to Run the Project
1. Download or clone this repository
2. Open the project folder
3. Double-click on `index.html`
4. The project will open in your web browser
## Application Workflow
1. User signs up or logs in  
2. Products are displayed on the homepage  
3. User adds products to cart  
4. Total price updates automatically  
5. User places order  
6. Order confirmation message is displayed
